Основы веб-разработки

2 недели
40 часов Начальный

Начните свой путь в веб-разработке с фундаментальных основ. Изучите HTML5, CSS3 и базовый JavaScript, создавая адаптивные и доступные веб-страницы.

Приобретаемые навыки:

  • Семантическая верстка HTML5
  • Современный CSS: Flexbox, Grid, анимации
  • Основы JavaScript: синтаксис, DOM, события
  • Адаптивный и мобильный дизайн
  • Доступность (a11y) и валидация
  • Основы работы с Git

Практические проекты:

  • Создание персонального портфолио
  • Адаптивный лендинг для продукта
  • Интерактивная форма с валидацией
